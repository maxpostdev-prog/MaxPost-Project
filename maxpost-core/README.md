# maxpost-core

Permanent WordPress functionality for the MaxPost platform.

## Responsibilities

- Custom Post Types and taxonomies
- Software metadata and validation
- WordPress Media Library integration
- REST API
- Updates and What's New
- Recommendations and promotions
- Cache invalidation
- Migrations and compatibility helpers

The plugin must preserve all platform data when the active WordPress theme changes.

See [`docs/08-MaxPost-Core.md`](../docs/08-MaxPost-Core.md).
