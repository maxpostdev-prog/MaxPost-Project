# maxpost-theme

Presentation layer for `maxpost.dev`.

## Responsibilities

- WordPress templates
- Design tokens and components
- Responsive layouts
- Accessibility of the public interface
- Front-end search, filters and galleries
- Rendering data supplied by `maxpost-core`

## Non-responsibilities

This directory must not own permanent data, Custom Post Types, taxonomies, REST business logic, licensing, update logic or migrations.

See [`docs/07-WordPress-Theme.md`](../docs/07-WordPress-Theme.md).
