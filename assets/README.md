# Assets

Shared visual assets for the MaxPost platform.

## Structure

- `logos/` — approved brand marks and lockups
- `icons/` — product and interface icons
- `mockups/` — website and application mockups
- `wireframes/` — exported layout blueprints
- `screenshots/` — approved product screenshots
- `diagrams/` — architecture and process diagrams

Only source-of-truth assets belong here. Temporary exports and generated build files must not be committed.
