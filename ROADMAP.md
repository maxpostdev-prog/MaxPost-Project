# MaxPost Roadmap

## Phase 0 — Foundation

- Define product vision
- Define brand and design system
- Split WordPress architecture into `maxpost-theme` and `maxpost-core`
- Prepare repository standards

## Phase 1 — Website MVP

- Build `maxpost-core`
- Build `maxpost-theme`
- Add software catalogue
- Add software detail pages
- Add screenshots and downloads

## Phase 2 — Platform API

- Software API
- Updates API
- What's New API
- Recommendations API
- Promotions API

## Phase 3 — First Utilities

- MP Folder Creator
- MP Bulk Rename
- MP ZIP Password Generator

## Phase 4 — Growth

- Multilingual website
- Analytics
- Internal recommendations
- Partner placements
- Donation support

## Phase 5 — Ecosystem

- Shared desktop UI library
- Common updater
- Common settings system
- Shared translation files

> Build small. Finish each stage before starting the next.
